# プロジェクト概要

これはvisionOSアプリ「太陽系シミュレータ」のプロジェクトです。
ユーザーはImmersive SpaceやVolumeで太陽系のシミュレーションを体験でき、天体にタッチすることで説明を見ることができます。

## 技術スタック

- Swift
- SwiftUI
- RealityKit
